# Selected Solutions to *Real Analysis*

## Basic Informations

This repository is my personal selected solutions to *Real Analysis*. Since most of those books were read by myself, correctness might not be guaranteed and this repository is for reference only.

Solutions were made using [TexMacs](https://www.texmacs.org/), saved as `.tm` files. May switch to $LaTeX$ in the future.

**NOTICE**: The propositions, theorems, corollaries, etc...in *italic form* are all from *Real Analysis (fifth edition)*, don't get confused!
